설치하기 #

npm #

$ npm install vue3-naver-maps --save

Yarn #

$ yarn add vue3-naver-maps

TypeScript를 이용한 프로젝트를 진행하고 있으시다면, @types/navermaps를 추가 설치 해주세요

설정 #

Naver Cloud Platform에서 지도 서비스 신청 후 발급된 Client ID 키가 필요합니다.

설치에 관한 자세한 옵션은 설치옵션 API를 참조해주세요

SSR

SSR 모드 기능을 지원합니다. 아래 예시와 똑같이 진행하면 SSR 모드 에서도 사용 가능합니다.

예시 #

// main.js
import { createApp } from "vue";
import naver from "vue3-naver-maps";

const app = createApp(App);

app.use(naver, { clientId: "your clientId" }).mount("#app");

<!-- your component -->
<template>
  <naver-maps> ... </naver-maps>
</template>

<script>
import { NaverMaps } from "vue3-naver-maps";

export default {
  components: { NaverMaps },
  setup: () => {},
};
</script>
이 문서에 대한 의견이 있으시거나, 문제를 발견하셨으면 제보해주세요!
