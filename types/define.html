타입정의 #

vue3-naver-maps에서 자체적으로 제공하는 naverV3 타입 정의 입니다.

initLayer #

지도 레이어 생성에 사용 됩니다.

type initLayer =
  | "BACKGROUND"
  | "BACKGROUND_DETAIL"
  | "BICYCLE"
  | "CADASTRAL"
  | "CTT"
  | "HIKING_TRAIL"
  | "PANORAMA"
  | "POI_KOREAN"
  | "TRANSIT"
  | "KOREAN"
  | "ENGLISH"
  | "CHINESE"
  | "JAPANESE";

mapOptions #

naver.maps.MapOptions를 상속 받아 사용합니다. 지도 옵션 생성에 사용 됩니다.

interface mapOptions extends naver.maps.MapOptions {
  latitude?: number;
  longitude?: number;
}

htmlIcon #

htmlIcon을 사용하는 marker의 초기 size와 anchor를 설정합니다.

interface htmlIcon {
  size: {
    width: number;
    height: number;
  };
  anchor: {
    x: number;
    y: number;
  };
}
이 문서에 대한 의견이 있으시거나, 문제를 발견하셨으면 제보해주세요!
